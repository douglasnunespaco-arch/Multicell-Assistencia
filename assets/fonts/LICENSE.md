# Fontes — Multi Cell Assistência Técnica

- **Sora** — SIL Open Font License 1.1 — https://fonts.google.com/specimen/Sora
- **Manrope** — SIL Open Font License 1.1 — https://fonts.google.com/specimen/Manrope

Arquivos self-hosted em formato WOFF2 (latin subset), pesos:
- Sora: 400, 600, 700
- Manrope: 400, 500, 600, 700

Ambas as fontes são open-source com licença permissiva para uso comercial
(SIL OFL 1.1). Nenhuma requisição externa é feita em runtime.
